Scrabble Game - HW5
Luke MacVicar
GUI Programming I

Description:
This project implements a full 15x15 Scrabble board game using jQuery UI for drag-and-drop functionality.

Features Implemented:
- The full scrabble board 15x15 grid is verified and playable.
- Tiles can be dragged from the rack to the board.
- Automatic score calculation including Double/Triple Letter and Word scores.
- Tiles are drawn randomly based on standard Scrabble rules.
- Checks for straight lines, connectivity, and dictionary validity.

How to Run:
Open index.html in a web browser.
